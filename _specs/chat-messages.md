# Chat Messages

## Описание

Просмотр сообщений в рамках выбранного чата. При клике на чат в списке загружаются и отображаются сообщения этого чата в хронологическом порядке в компоненте `ChatWindow`.

## Мотивация

Коммерческому представителю необходимо видеть историю переписки с продавцом при выборе чата из списка. На данный момент компонент `ChatWindow` существует, но не загружает сообщения с сервера. Нужно связать выбор чата с загрузкой и отображением его сообщений.

## Требования

### API

- Метод: `GET`
- URL: `{{BASE_URL}}chats/{{chat_id}}/messages`
- Авторизация: заголовок `Authorization: Bearer <token>` (подставляется автоматически через `httpClient`)
- Входящих параметров нет (payload отсутствует)
- Ответ: массив объектов сообщений в формате
```
[
    {
        "code": "1",
        "text": "asdasd",
        "createdAt": "2026-04-14T06:45:13.633Z",
        "userCode": "1",
        "userName": "User User"
    }
]
```

### Загрузка данных

- При выборе чата в `ChatList` (клик на элемент) загружаются сообщения выбранного чата
- Полученные сообщения сохраняются в `chatStore`
- При смене активного чата загружаются сообщения нового чата

### Отображение

- Сообщения отображаются в компоненте `ChatWindow` в хронологическом порядке (от старых к новым)
- Каждый элемент сообщения содержит:
  - Текст сообщения
  - Имя автора (`userName`)
  - Дата/время в формате:
    - `HH:mm` — если сообщение отправлено сегодня
    - `DD.MM.YYYY HH:mm` — если сообщение отправлено в другой день
- При загрузке сообщений отображается индикация загрузки
- При ошибке запроса отображается сообщение об ошибке

## Затрагиваемые области

- API-слой (новая функция получения сообщений чата)
- Тип `Message` в `src/types/index.ts` (приведение к формату API)
- `chatStore` (загрузка сообщений, состояния загрузки)
- Компонент `ChatWindow` (отображение списка сообщений)
- Компонент `ChatList` (инициация загрузки сообщений при выборе чата)

## Зависимости

- `httpClient` — для выполнения авторизованных запросов к API
- `chatStore` — для хранения сообщений и активного чата
- Существующий тип `Message` из `src/types/index.ts`

## Вне скоупа

- Отправка новых сообщений
- Обновление сообщений в реальном времени (WebSocket)
- Пагинация / подгрузка истории при скролле
- Вложения (файлы, изображения)
- Индикация прочтения / доставки сообщений
- Группировка сообщений по дате
